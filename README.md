# Kütüphane Yönetim Sistemi

Bu proje, veritabanı dersi kapsamında geliştirilen bir kütüphane yönetim sistemidir.

## Proje Hakkında
Bu sistem ile kütüphanedeki kitaplar, üyeler ve gerçekleştirilen işlemler yönetilebilmektedir.

## Kullanılan Teknolojiler
* MySQL
* [Buraya kullandığın dili/aracı yaz, örneğin: PHP/Python]

## Nasıl Çalıştırılır?

database/kutuphane_dump.sql dosyasını MySQL Workbench üzerinden kutuphane adında bir veritabanına Import edin.

Web dosyalarını XAMPP/WAMP sunucunuzun htdocs klasörüne taşıyın.

config.php veya veritabanı bağlantı kodlarının bulunduğu dosyada veritabanı kullanıcı adını ve şifresini kendinize göre güncelleyin.
